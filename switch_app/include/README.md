# `switch_app/include`

Headers del proyecto Switch.

La estructura replica `source/`:

- `domain/`: declaraciones de reglas puras.
- `application/`: casos de uso.
- `adapters/`: puertos hacia MTP, SD, saves, UI, instalador.
- `ports/`: interfaces internas para mantener arquitectura hexagonal.
- `ui/`: componentes gráficos.
