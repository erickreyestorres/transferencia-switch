# Entrada de recursos visuales

Deja en esta carpeta los iconos e imágenes originales generados para la interfaz.

Formatos preferidos:

- SVG para iconos vectoriales.
- PNG con fondo transparente si no existe SVG.
- Sprite sheets PNG para animaciones.
- Archivo JSON o texto con el nombre y significado de cada recurso, si está disponible.

No es necesario cambiar tamaños ni nombres antes de copiarlos. Después de revisarlos se
normalizarán y exportarán a las carpetas finales de la aplicación.
