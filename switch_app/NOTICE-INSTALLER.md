# Referencias del instalador

El receptor NSP fue implementado para Transferencia Switch sobre las APIs públicas de
libnx (`ncm`, `ns`, `fs` y servicios IPC). Para contrastar el flujo de placeholders,
CNMT y registro de aplicaciones se estudió Awoo Installer y el trabajo original de
Adubbz/Tinfoil. Los archivos consultados declaran licencia MIT; el repositorio de Awoo
se conserva únicamente fuera del producto, en el directorio local de investigación.

- Awoo Installer: https://github.com/Huntereb/Awoo-Installer
- libnx: https://github.com/switchbrew/libnx

No se incorporó código GPL del repositorio de referencia al árbol de la aplicación.
