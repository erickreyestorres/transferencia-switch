# `tools/homebrew-icon`

Archivos auxiliares usados para integrar el icono del Homebrew Menu.

- `aplicar_icono.ps1`: script auxiliar.
- `transferencia-switch-icon.patch`: patch histórico del cambio.
- `README_ICONO.md`: notas originales del proceso.

`switch_app/icon.jpg`
